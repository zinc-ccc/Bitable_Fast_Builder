# Bitable-Fast-Builder 项目介绍

## 1. 项目概述
**Bitable-Fast-Builder** 是一个基于飞书（Feishu/Lark）平台的自动化工具，旨在通过**自然语言描述**快速生成并搭建规范化的**多维表（Bitable）**。

其核心目标是解决在业务搭建过程中，从沟通需求到实际建表之间的繁琐过程。通过集成飞书机器人，用户只需在对话框中发送业务描述，系统即可自动解析字段、应用命名规范，并一键完成表格的物理创建。

## 2. 核心功能
*   **需求解析**：利用 AI 逻辑解析自然语言，识别业务实体和所需字段。
*   **字段规范化**：自动根据“看板美观度”与“可读性”准则设计表头（例如：推荐使用 `入职日期:计划` 而非复杂符号）。
*   **自动化搭建**：通过飞书 API 自动创建多维表、数据表及配置字段。
*   **成员同步 (HECS 4.1)**：支持从飞书群组自动拉取成员信息、识别组织架构，并同步至配置中心表中。

## 3. 技术架构
*   **开发语言**：Python 3
*   **核心 SDK**：`lark-oapi` (飞书开放平台官方 SDK)
*   **配置管理**：YAML 格式配置文件
*   **通信模式**：支持 WebSocket 长连接模式，无需公网 IP 即可接收事件回调。
